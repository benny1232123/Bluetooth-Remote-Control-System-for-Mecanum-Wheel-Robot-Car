# 蓝牙遥控麦克纳姆轮小车控制系统

本项目基于 STM32F103ZET6，实现了麦克纳姆轮全向移动、PID 闭环控制、OpenMV 视觉追踪、步进电机升降台及舵机云台控制，并支持蓝牙遥控。

## 串口配置
- **UART5 (默认蓝牙)**: TX=PC12, RX=PD2, 9600 波特率 (由宏 `BT_USE_UART5 1` 控制)
- **USART1 (备选蓝牙)**: TX=PA9, RX=PA10 (需设置 `BT_USE_UART5 0`)
- **USART2 (步进电机)**: TX/RX, 115200 波特率 (驱动 Emm_V5 步进电机)
- **USART3 (陀螺仪)**: TX=PB10, RX=PB11, 9600 波特率
- **UART4 (OpenMV)**: TX=PC10, RX=PC11, 115200 波特率

## 蓝牙指令汇总

### 行驶指令 (单字符)
- `F`: 前进
- `B`: 后退
- `L`: 左移
- `R`: 右移
- `A`: 左旋
- `D`: 右旋
- `S` 或 `X`: 停止
- `1..5`: 速度档位 (1 最慢, 5 最快)

### 升降台 (步进电机) 指令
- `U`: 上升 (慢速)
- `N`: 下降 (慢速)
- `Z`: 停止

### 载物盘 (舵机) 指令
- `Q`: 左转 (步进)
- `E`: 右转 (步进)
- `V`: 居中

### 爪子 (舵机) 指令
- `O`: 张开爪子 (慢速逼近)
- `P`: 闭合爪子 (慢速逼近)

### 云台 (舵机) 指令
- `J`: 左转 (步进)
- `K`: 右转 (步进)
- `C`: 居中

## 行为说明
- **按下/松开控制**: 行驶键使用按下/松开模式，按住时车辆保持运动，松开后立即停止.
- **指令超时**: 如果蓝牙端漏发松开包，0.5 秒内无新指令 (`BT_TIMEOUT_TICKS = 5` @10Hz) 时小车也会自动停止.
- **心跳**: MCU 每 0.2 秒 (`BT_HEARTBEAT_TICKS = 2` @10Hz) 发送一次 'H', 用于保活与调试.

## 按钮型（按下保持）模式说明
- 为实现“按下保持”行为，固件采用约定：**小写字符表示按下（开始动作），对应的大写字符表示松开（停止动作）**。
- 例如：发送 `f`（小写）表示按下前进按钮，松开时发送 `F`（大写）表示停止前进。
- 兼容性：如果你的蓝牙终端不能区分按下/松开，请在 App 中配置“按下发送小写，松开发送大写”。

## 按钮配置模板（按下 / 释放 / 长按）
下面为常见手机 App 或自定义遥控的完整配置模板，按步骤设置后直接可用。

1) 总规则
- 小写 = 按下（Press，开始动作）
- 大写 = 释放（Release，停止动作）
- 长按 = 重复发送对应小写（建议间隔 120–200ms，推荐 150ms）以保持动作稳定

2) 摇杆 + 按键最终映射（直接套用）
- 左摇杆（移动）:
	- 上: Press `f`  Release `F`  Long-press 重复 `f`
	- 下: Press `b`  Release `B`  Long-press 重复 `b`
	- 左: Press `l`  Release `L`  Long-press 重复 `l`
	- 右: Press `r`  Release `R`  Long-press 重复 `r`
- 右摇杆（旋转）:
	- 左: Press `a`  Release `A`  Long-press 重复 `a`
	- 右: Press `d`  Release `D`  Long-press 重复 `d`
- 停止键: 单点 `s` 或 `S`

- 升降台:
	- 上升: Press `u`  Release `U`  Long-press 重复 `u`
	- 下降: Press `n`  Release `N`  Long-press 重复 `n`
	- 停止: 单点 `z` 或 `Z`

- 载物盘（平台）:
	- 左转: Press `q`  Release 可空  Long-press 重复 `q`
	- 右转: Press `e`  Release 可空  Long-press 重复 `e`
	- 居中: 单点 `v` 或 `V`

- 爪子:
	- 打开: 单点 `o`
	- 关闭: 单点 `p`

- 云台:
	- 左转: Press `j`  Release 可空  Long-press 重复 `j`
	- 右转: Press `k`  Release 可空  Long-press 重复 `k`
	- 居中: 单点 `c` 或 `C`

3) 旋转类补充映射（已在固件中支持，建议加入 App 的第二页或长按层）
- 云台居中: Press `c`

4) 关键使用方式
- 行驶类按键必须配置成 Press/Release 两个事件：Press 发送小写字母，Release 发送对应大写字母.
- 如果你的 App 只能发单次字符，建议换成支持按下/松开回调的遥控器，否则无法严格做到“按多久走多久、松开就停”.

5) 在常用 App 中的具体设置示例
- Serial Bluetooth Terminal（或类似支持 Press/Release/Long-press 的 App）:
	- 新建按钮 “前进” → Press 字段输入 `f`，Release 字段输入 `F`，Long-press 打开并设置 Payload=`f`，Interval=150ms。
	- 其他按钮按上表配置。
- BLE Remote / Remote Control 之类（有单独长按/重复选项的）:
	- Press = 小写字符，Release = 大写字符，Long-press = 重复发送小写，间隔 120–200ms。
	- 如果 App 支持“多页按钮”，第一页放 1~12 键，第二页放补充键 `c`。
	- 爪子按钮建议：第 9 键发 `p` 关闭，第 10 键发 `o` 张开。

### 速度控制（固件支持 + / -）
- 我已在固件中加入对 `+`（加速）和 `-`（减速）的支持：
	- 接收到 `+`（小写或大写均可）会把速度档位 `bt_speed_level` 增 1（上限 5 档，但内部实现为 0..4），并通过蓝牙回显当前档位字符 `1..5`；
	- 接收到 `-` 会把速度档位减 1 并回显当前档位。
- 建议在你的 12 键中将第 12 键设置为：
	- Press: `+`  Long-press: 重复 `+`（Interval=150ms） 用于加速；
	- 如需减速，把某个备用按键设置为 `-`，或把长按停止键设置为发送 `-`。


6) 调试与优化
- 如果按住无响应：将长按重复间隔调短至 100–150ms；或在 App 启用“持续发送”。
- 若出现抖动或重复短脉冲：在 App 端启用去抖 30–80ms，或我可以在固件侧加入 50ms 软件去抖。

## OpenMV 通信协议
- 帧格式: `[标志符]X数据X Y数据Y` (例: `F132X45Y`)
- 标志符: `A~G` 表示识别到目标，`Q` 表示未识别到目标。
- 当接收到 `Q` 且追踪开启时，小车将强制停止电机输出。
