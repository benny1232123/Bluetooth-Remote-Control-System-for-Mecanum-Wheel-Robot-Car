# 蓝牙遥控麦克纳姆轮小车控制系统

本项目基于 STM32F103ZET6，实现了麦克纳姆轮全向移动、PID 闭环控制、OpenMV 视觉追踪、步进电机升降台及舵机云台控制，并支持蓝牙遥控。

## 串口配置
- **UART5 (默认蓝牙)**: TX=PC12, RX=PD2, 9600 波特率 (由宏 `BT_USE_UART5 1` 控制)
- **USART1 (备选蓝牙)**: TX=PA9, RX=PA10 (需设置 `BT_USE_UART5 0`)
- **USART2 (步进电机)**: TX/RX, 115200 波特率 (驱动 Emm_V5 步进电机)
- **USART3 (陀螺仪)**: TX=PB10, RX=PB11, 9600 波特率
- **UART4 (OpenMV)**: TX=PC10, RX=PC11, 115200 波特率

## 蓝牙指令汇总

### 行驶指令 (单字符，按下保持)
- `F`/`B`: 前进/后退
- `L`/`R`: 左移/右移
- `A`/`D`: 左旋/右旋
- `S` 或 `X`: 停止
- `1`..`5`: 速度档位 (1 最慢, 5 最快)
- `+`/`-`: 加速/减速

### 升降台 (步进电机，按下保持)
- `U`: 上升
- `N`: 下降
- `Z`: 停止

### 云台 (舵机，点动步进)
- `J`: 左转一步
- `K`: 右转一步
- `C`: 居中
- `G`: 左转 180°
- `I`: 右转 180°

### 载物盘 (舵机，点动步进)
- `Q`: 左转一步
- `E`: 右转一步
- `V`: 居中
- `T`: 左转 125°
- `Y`: 右转 125°

### 爪子 (舵机，点动即到位)
- `O`: 张开 (到基准位)
- `P`: 闭合 (夹紧)

## 行为说明
- **行驶/升降台使用按下保持模式**：小写=按下开始，大写=松开停止。例如 `f` 前进，`F` 停止前进。
- **云台/载物盘/爪子为点动模式**：每次收到指令执行一步动作，松开不重复。
- **指令超时**: 0.5 秒无新指令时自动停止行驶。
- **心跳**: MCU 每 0.2 秒发送 `H` 用于保活。

## OpenMV 通信协议
- 帧格式: `[标志符]X数据X Y数据Y` (例: `F132X45Y`)
- 标志符: `A~G` 表示识别到目标，`Q` 表示未识别到目标。
- 当接收到 `Q` 且追踪开启时，小车将强制停止电机输出。
