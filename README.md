# 蓝牙遥控麦克纳姆轮小车控制系统

本项目基于 STM32F103ZET6，实现了麦克纳姆轮全向移动、PID 闭环控制、OpenMV 视觉追踪、步进电机升降台及舵机云台控制，并支持蓝牙遥控。

## 串口配置
- **UART5 (默认蓝牙)**: TX=PC12, RX=PD2, 9600 波特率 (由宏 `BT_USE_UART5 1` 控制)
- **USART1 (备选蓝牙)**: TX=PA9, RX=PA10 (需设置 `BT_USE_UART5 0`)
- **USART2 (步进电机)**: TX/RX, 115200 波特率 (驱动 Emm_V5 步进电机)
- **USART3 (陀螺仪)**: TX=PB10, RX=PB11, 9600 波特率
- **UART4 (OpenMV)**: TX=PC10, RX=PC11, 115200 波特率

## 蓝牙指令汇总

> **大小写约定**：`小写`=发送小写字母触发动作，`大写`=发送大写字母停止动作。
> 点动类（云台/载物盘/爪子）只发小写即可，大写会被忽略。

### 行驶指令 (按下保持)
| 发送 | 动作 |
|------|------|
| `f` / `F` | 前进 / 停止前进 |
| `b` / `B` | 后退 / 停止后退 |
| `l` / `L` | 左移 / 停止左移 |
| `r` / `R` | 右移 / 停止右移 |
| `a` / `A` | 左旋 / 停止左旋 |
| `d` / `D` | 右旋 / 停止右旋 |
| `s` 或 `x` | 立即停止 |
| `1`~`5` | 速度档位 (1 最慢, 5 最快) |
| `+` / `-` | 加速 / 减速 |

### 升降台 (步进电机，按下保持)
| 发送 | 动作 |
|------|------|
| `u` / `U` | 上升 / 停止上升 |
| `n` / `N` | 下降 / 停止下降 |
| `z` | 急停 |

### 云台 (舵机，点动)
| 发送 | 动作 |
|------|------|
| `j` | 左转一步 |
| `k` | 右转一步 |
| `c` | 居中 |
| `g` | 左转 180° |
| `i` | 右转 180° |

### 载物盘 (舵机，点动)
| 发送 | 动作 |
|------|------|
| `q` | 左转一步 |
| `e` | 右转一步 |
| `v` | 居中 |
| `t` | 左转 125° |
| `y` | 右转 125° |

### 爪子 (舵机，点动)
| 发送 | 动作 |
|------|------|
| `o` | 张开 (到基准位) |
| `p` | 闭合 (夹紧) |

### 自动任务 / 摄像头测试
| 发送 | 动作 |
|------|------|
| `m` | 开始自动取放流程：升降台向下→爪子闭合→升降台向上→云台旋转180°→升降台向下→爪子打开→升降台向上→云台返回 |
| `w` | 切换摄像头测试模式（开关）。开启后摄像头检测到目标即前进1秒 |

## 行为说明
- **行驶/升降台**：小写=按下开始动作，大写=松开停止。
- **云台/载物盘/爪子**：每次收到小写指令执行一次，松开发大写不产生动作。
- **摄像头测试模式 (w)**：按一次开启，再按一次关闭。开启后摄像头看到红色目标自动前进1秒，用于验证摄像头与主控通信。
- **超时保护**: 0.3 秒无新指令时自动停止行驶。
- **心跳**: MCU 每 0.2 秒发送 `H` 用于保活。

## OpenMV 通信协议
- 帧格式: `[标志符]X数据X Y数据Y` (例: `F132X45Y`)
- 标志符: `A~G` 表示识别到目标，`Q` 表示未识别到目标。
- 当接收到 `Q` 且追踪开启时，小车将强制停止电机输出。
